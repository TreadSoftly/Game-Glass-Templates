## Active Reticle
- WarThunder/Active/Dre4d Skull Sight.blk
- WarThunder/Active/sight_1.blk (identical copy)
Source PNG: WarThunder/Source/Dre4d Sight.png
Baselines: WarThunder/Baselines/
## Install (PC)
Copy Dre4d Skull Sight.blk (or sight_1.blk) into:
Documents\My Games\WarThunder\Saves\<userId>\production\UserSights\all_tanks\
Restart War Thunder and select Dre4d Skull Sight (or User sight for sight_1).
## Rangefinder / Overlay
The full-screen lines and top number bar are the built-in shot-distance scale overlay.
We keep it hidden via shotDistScaleOffset:i=-100000 in global.blk.
See WarThunder/Config/global.blk and WarThunder/Notes/reticle-notes.md.
